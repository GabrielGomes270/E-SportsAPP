# 🎮 E-SportsAPP - API

API desenvolvida em **ASP.NET Core 8** com banco de dados **MySQL**, utilizando Docker para facilitar a configuração e execução do projeto.

---

## 🚀 Tecnologias

- C#
- ASP.NET Core 8
- Entity Framework Core
- MySQL
- Docker + Docker Compose
- Swagger para documentação
